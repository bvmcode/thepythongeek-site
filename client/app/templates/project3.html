My Weather Station Data

My Ambient Weather Station is located in Atco, NJ. Below is current data and data for the last 12 hours. More to come as this data is collected over time.

I created a Python package for ingesting API data for Ambient Weather Stations, available on GitHub and PyPI.

The AWS infrastructure I built around data collection is on GitHub and includes:

  • Docker image with scheduled data collection
  • Terraform script that provisions:
    • ECR repository for the image
    • EC2 instance that pulls and runs the image
    • RDS instance for data persistence

Station data is also collected on Weather Underground.
